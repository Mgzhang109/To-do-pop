# To-do Pop

一个用“重要 x 紧急”四象限整理待办事项的小软件原型。任务显示为彩色气泡，拖动气泡可以改变紧急/重要坐标，点击气泡会播放爆破音效和粒子飞溅，并自动进入“已完成DONE”收纳仓库。

## 当前功能

- 重要紧急四象限坐标面
- 彩色待办气泡，气泡大小可调
- 大项目气泡，内部显示多个小任务气泡
- 新建、编辑、删除、恢复任务
- 点击完成后进入“已完成DONE”
- 爆炸粒子效果和 Web Audio 爆破音效
- 自动同步清单表格：TO DO / DOING / DONE
- 本地 `localStorage` 持久化

## 本地运行

```bash
npm install
npm run dev
```

打开终端里显示的本地地址即可使用。
